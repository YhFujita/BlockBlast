# Google Apps Script (GAS) デプロイ手順

1.  **スプレッドシートを開く**
    - 新しいスプレッドシートを作成するか、既存のスプレッドシートを開きます。

2.  **Apps Script エディタを開く**
    - メニューの「拡張機能」 > 「Apps Script」をクリックします。

3.  **コードの貼り付け**
    - 開いたエディタの `コード.gs` (または `Code.gs`) の内容をすべて削除します。
    - ローカルに作成された `Code.js` (このフォルダにあります) の内容をすべてコピーして、エディタに貼り付けます。
    - `Ctrl + S` で保存します（プロジェクト名は適当でOKです）。

4.  **デプロイ**
    - 右上の「デプロイ」ボタン > 「新しいデプロイ」を選択します。
    - 「種類の選択」の歯車アイコンをクリック > 「ウェブアプリ」を選択します。
    - 設定項目：
        - **次のユーザーとして実行**: `自分`
        - **アクセスできるユーザー**: `全員` (**重要！これを選択しないと動きません**)
    - 「デプロイ」ボタンをクリックします。
    - 「アクセスを承認」が求められた場合：
        - アカウントを選択 > 「詳細」 > 「(安全ではないページ)に移動」 > 「許可」の順に進んでください。

5.  **URLの取得**
    - デプロイが完了すると「ウェブアプリのURL」が表示されます。
    - このURL ( `https://script.google.com/macros/s/.../exec` ) をコピーしてください。
    - **チャットでこのURLを教えてください**（エディタのコードに埋め込む必要があります）。
